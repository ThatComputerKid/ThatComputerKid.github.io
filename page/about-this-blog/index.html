About This Blog

2020-05-29

This isn’t actually a blog, really. There is no structure, or subject, though I’ll bet it’s going to revolve around tech. I like to think of this as a brain dump that the whole world can see.

On the back end this blog is a static site. Made with Hugo and the bilberry theme.

It was very fun and easy to make. It’s the first website I’ve made in the last 5 years-ish. When I about 7 I did make a dumb website that you don’t want to check out, with Wordpress. When I decided I wanted a slice of the Internet pie, I knew that it didn’t need to be a big bulky Wordpress so I started investigating static sites. I settled on Hugo and here we are!
