The Lounge IRC

2020-05-29

The Lounge IRC client is a self-hosted web IRC client.

Basically it’s an IRC client that will run on your server or always connected computer so that it never misses a message. You can then connect to the Lounge through any web browser and read those messages. It is beautifully designed, has the features of any modern chat client, and is easy to deploy.

I use IRC a lot for Jupiter broadcasting, support, and more.

I’m going to tell you how I set it up if you don’t mind.

My main server is a Pine A64 single board computer with an ARM 64 CPU. I prefer to host my services using docker. The Pine A64 is ARM it unfortunately can’t run the official Lounge IRC docker image. Luckily Linux Server has made a docker image for it that runs on ARM. So what I need to do is run this script

docker run –detach
–name thelounge
–publish 9000:9000
–volume ~/.thelounge:/var/opt/thelounge
–restart always
thelounge/thelounge:latest

This will pull down the latest image and run it in the background. It will also make a directory in your /home/username called thelounge. This Directory holds your config.js for the lounge. I recommend you read and edit this config to your liking. Unless you plan to have strangers using your Lounge IRC instance, I strongly recommend changing public: true, to public: false,. Once you have all that finished, you can add your user to The Lounge by running docker exec --user root -it //thelounge thelounge add YoUrUsErNaMe Then it will ask you for a password. MAke sure you change YoUrUsErNaMe to something more logical :)

You should now be good to go!

Visit 192.168.X.X:9000 replacing the X with your server’s IP. Now and your IRC channels!
