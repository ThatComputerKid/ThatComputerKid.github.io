The Lounge IRC

2020-05-29
The Lounge IRC client is a self-hosted web IRC client. Basically it’s an IRC client that will run on your server or always connected computer so that it never misses a message. You can then connect to the Lounge through any web browser and read those messages. It is beautifully designed, has the features of any modern chat client, and is easy to deploy.
